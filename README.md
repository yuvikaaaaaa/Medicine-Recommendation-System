# 📖 About the Project

Healthcare decisions often begin with understanding symptoms, but identifying the right condition and finding reliable guidance can be confusing and time-consuming.

**CuraAI** is an AI-powered healthcare recommendation platform that combines Machine Learning with a modern full-stack architecture to assist users in understanding their health better.

Users can securely log in, enter their symptoms, receive disease predictions, explore personalized medications, diet plans, precautions, workout suggestions, visualize health analytics, track previous predictions, and download professional PDF health reports.

The project demonstrates the integration of **Machine Learning, FastAPI, React, PostgreSQL, and Google OAuth** into a complete production-style healthcare application.

# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/yuvikaaaaaa/CuraAI.git
```

---

## Backend Setup

```bash
cd backend

pip install -r requirements.txt

uvicorn main:app --reload
```

---

## Frontend Setup

```bash
cd frontend

npm install

npm run dev
```
